**Structural
Travel Fixed → Store Loop 전이율**
→ 전체 루프의 첫 연결이 실제로 형성되는지 확인

**Store Loop → Confirm Loop 전이율**
→ 저장 이후 확인 루프로 자연스럽게 이어지는지 확인

**`Backflow`**
 → 확인 루프에서 다시 저장 루프로 돌아가는 역류 정도 확인

**`Total_LCP`**
→ 전체 준비 구조가 실제로 닫히는지 보는 핵심 구조 지표

**Lagging
`Total_LSI`**
→ 전체 루프 성공 확률에 반복 탐색과 역류를 함께 반영한 종합 안정성 지표

**Prepare Action → Reuse 전이율**
→ 실제 준비 행동 이후 다시 서비스 재사용으로 이어졌는지 확인

**새로운 Store Loop 발생률**
→ 다음 여행 준비에서 다시 체크리스트 생성 및 탐색 루프로 진입하는지 확인
