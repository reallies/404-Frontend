## 1. `/` — 홈 / 랜딩 (`HomePage`)

### 카피 추가 제안
- [ ] 아래 문구 계열을 랜딩에 반영
  - "AI가 알아서 여행지에 따라 필요한 준비항목을 제공해줍니다. 여러분은 필요한 것만 저장하고 준비하면 됩니다."
  - "정보를 찾기 위한 반복 탐색 없이"

### 멘트 수정
- [ ] **1/ 멘트 수정**: "그만" → **"그만!"** (느낌표 추가)
  - 참고 이미지: `보우마스터/image.png`
- [ ] **2/ 멘트 강조** 처리 필요
  - 참고 이미지: `보우마스터/image 1.png`
- [ ] **3/ 멘트 수정 + 노란색 하이라이트 강조**
  - 참고 이미지: `보우마스터/image 2.png`
  - 제목: `준비는 쉽게, 여행은 완벽하게 ✈️`
  - 내용:
    ```
    여행 준비는 생각날 때마다 조금씩 늘어나기 마련이죠.
    목적지와 일정에 맞는 항목을 골라 나의 체크리스트로 모아두면,
    놓치기 쉬운 준비물도 한눈에 정리됩니다.
    출발 전까지 리스트로 점검하며 완벽한 여행을 준비해보세요!
    ```
- [ ] **4/ 멘트 정렬 수정**
  - 참고 이미지: `보우마스터/image 3.png`
- [ ] **5/ 이미지 변경 + 카피 수정**
  - 참고 이미지: `보우마스터/image 4.png`, `보우마스터/Group 355.png`
  - 카피:
    ```
    ~ 정리하고,
    체크리스트로 출발 전까지 한눈에 확인할 수 있는
    여행 준비 서비스입니다.
    ```

- [ ] **아래 두 가지 중 확정 필요**

  - (B) 실제 총 4단계 → 모든 페이지의 step 표기를 **총 step4로 수정**
